===============================================
DEPLOYMENT STEPS
===============================================

1. Upload to VM server:
   - Copy entire d:\MES folder to VM
   - OR use git clone

2. Install dependencies:
   cd /opt/MES  (Linux)
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements_new.txt

3. Configure firewall:
   sudo ufw allow 8501/tcp

4. Set up service:
   See VM_DEPLOYMENT_GUIDE.md for systemd/NSSM setup

5. Test:
   - Access from VM: http://localhost:8501
   - Access from network: http://[VM_IP]:8501

===============================================
BACKUP STRATEGY
===============================================

Automatic:
  - After every data entry save
  - After every process completion
  - Keeps last 30 backups
  - Location: backups/ folder

Manual:
  - "Create Backup Now" button in Reports tab
  - Backup history viewer
  - Database size display

Recommended:
  - Weekly: Copy backups to external USB drive
  - Monthly: Upload to cloud storage (Google Drive)
  - Before updates: Always create manual backup
